2.2 🔴 RS20 停产影响专题

Hirschmann RS20 系列（含 RS20-0400/0800/0900 全部变体）是其出货量最大的工业以太网接入交换机，已于 2023 年 12 月 31 日 全面停产。

#风险影响IE3100 如何化解

2.4 本章五大结论

1. 产品线活跃率

Cisco ≈ 100% vs Hirschmann ≈ 70–75%。RS20 停产和 BAT-C2 停售预告对 Hirschmann 产品线健康度构成实质性打击。

2. 最大替换窗口

RS20 → IE3100 是当前市场上最大的替换机会。数以万计在用设备面临备件、安全、带宽三重风险，IE3100 提供完整 1:1 升级路径。

3. 全线 GE/10GE

Cisco 全线千兆以上，Hirschmann 仍有大量百兆产品。在工业 4.0 时代，100Mbps 已是过时瓶颈。

4. OS 统一性

Cisco 全线 IOS-XE = 一套培训、一套工具、一套自动化脚本。Hirschmann 的 HiOS / Classic / HiLCOS 碎片化增加运维成本。

5. 安全不是"附加件"，是"内建能力"

Cisco 将 Cyber Vision、MACsec、ISE 微分段嵌入交换机本身；Hirschmann 需额外采购 EAGLE，增加成本和复杂度。

04 操作系统对比 — IOS-XE vs HiOS

工业交换机的价值远不止于硬件参数。真正决定运维效率、安全韧性与长期演进能力的，是底层操作系统。如果说硬件是交换机的"身体"，那操作系统就是它的"灵魂"。本章从六大维度展开深度对比：架构基因、管理接口、协议栈、可编程性、边缘计算、升级维护。

06 OT 安全深度分析

在 IT 世界，"安全是最后才加上去的螺栓"这句自嘲流传了三十年；遗憾的是，大部分 OT 网络至今仍在重复同样的错误。当一座焊装车间或半导体 FAB 被勒索软件锁死时，每小时损失可达 $250,000–$500,000。真正的问题不是"要不要做安全"，而是——安全能力应该长在网络骨架里，还是像创可贴一样贴在外面？

6.1 安全架构哲学：骨骼 vs. 盔甲

🦴 Cisco = 骨骼（嵌入式纵深防御）

安全与网络融为一体、无法剥离。每一台交换机都是传感器 + 执行点——数据平面即安全平面。

  • Cyber Vision Sensor 运行在 IOx 容器内，零额外硬件
  • ISE + TrustSec SGT 标签实现端口级微分段
  • MACsec-256 线速加密，零性能损失
  • Talos 400+ 研究员，24h 内发布 ICS 检测规则
  • SecureX/XDR 自动关联 → ISE 自动隔离 ≤30 秒

🛡️ Hirschmann = 盔甲（外挂式边界防护）

安全需要额外穿戴、可以卸下。交换机负责转发，安全由专用设备承担——经典边界防御思维。

  • EAGLE40 独立 DIN-Rail 防火墙，需单独采购部署
  • VLAN + ACL 手动分段
  • 交换机本身无 MACsec
  • Belden 安全团队规模 <20 人（估计）
  • 人工识别 + 手动修改规则：15–60 分钟
🏭 类比：当你保护的不是一个骑士而是一支军队时，"每个人都自带骨骼"显然比"给每个人定制一套盔甲"更经济、更敏捷。在 Purdue 模型 L2/L1 层存在数十个微小 Cell 的现代工厂中，嵌入式模型的覆盖密度和响应速度具有结构性优势。

A.4 专题：Layer 2 NAT — OT 网络的 IP 冲突终结者

在 OT（运营技术）网络中，OEM 机台出厂 IP 地址重复是一个普遍且棘手的难题。 多台相同型号的 PLC / 机器人控制器往往使用相同的默认 IP 段（如 192.168.1.0/24），而修改设备 IP 可能导致 OEM 质保失效或需要昂贵的重新调试。Cisco IOS XE 提供的 Layer 2 NAT 功能正是为此而生。

❶ 问题：OEM 机台 IP 重复

同一产线部署 5 台相同 CNC 控制器，每台出厂 IP 均为 192.168.1.10。 传统方式需逐台改 IP 或增加 L3 NAT 设备，成本与复杂度高。

❷ 传统 L3 NAT 的局限

L3 NAT 改变三层地址但同时改变 VLAN 域，导致广播/组播依赖的工业协议 （PROFINET DCP、EtherNet/IP 隐式消息）失效，且需要额外路由器/防火墙。

❸ L2 NAT 的创新

Cisco L2 NAT 在二层域内完成 IP 地址重写：数据帧不离开 VLAN， MAC 不变，仅重写 IP 头。对上位 SCADA/MES 系统来说，每台机台拥有唯一 IP， 而机台本身无需任何配置变更。

❹ 为何更适合 OT

✅ 不改 OEM 配置 → 质保无忧
✅ 不离开 VLAN → 组播/广播工业协议不受影响
✅ 无需额外设备 → 交换机内嵌处理
✅ 支持双向映射 → SCADA 下行指令同样透明转换

L2 NAT 核心价值： Cisco Layer 2 NAT 是当前工业交换机市场中唯一在二层域内解决 IP 冲突的方案。 对于拥有大量 OEM 重复 IP 设备的制造业客户，它可以显著降低网络改造成本、 保留 OEM 质保并维持工业协议正常通信——这是 Hirschmann HiOS 及其他竞品目前无法提供的差异化能力。
